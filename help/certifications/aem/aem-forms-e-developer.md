* Nivå: Expert (1-3 års erfarenhet)

* Leverans: Bearbetad online (kräver kameraåtkomst)

### Målgrupp {#intended-audience}

* AEM
* AEM utvecklare
* AEM konsulter
* AEM Architects

<br>

### Minimal upplevelse {#minimum-experience}

Du borde ha:

* 1-3 års erfarenhet av AEM Forms
* Minst fem års arbetslivserfarenhet inom IT-branschen
* Möjligheten att leda utvecklingen, konfigurationen, testningen och driftsättningen av skräddarsydda blankettlösningar
* Möjlighet att konfigurera ett formulär, omvandla affärskrav till arbetsflöden, integrera med externa system
* En förståelse för AEM Forms grundkomponenter

Dessutom bör du känna till följande tekniker och miljöer:

* Designa XDP-mallar, XFA-mallar
* Skapa PDF forms (statiskt/dynamiskt, interaktivt och icke-interaktivt)
* Schema eller XML-databindning
* Skapa formulärfragment
* Skapa ett adaptivt formulär
* Felsökning av komplexa formulär
* Konfigurera och skicka formulär
* Skapa och distribuera AEM för anpassningsbara formulär
* Använd formulärmodell
* Möjlighet att integrera med datatjänster
* Implementering av arbetsflöden
* Grundläggande förståelse för AEM formulär workbench (JEE) för att skapa långa, dynamiska processer och tjänster.
* Utveckla JavaScript clientlibs med GuideBridge API
* Baserat på sign v government vet CAC vilken AEM Forms-miljö som är stationär eller serverbaserad att använda
* Grundläggande förståelse av Java/J2EE och JavaScript för att skriva logik och ringa olika tjänster.
* Fyll i ett anpassat formulär i förväg
* Grundläggande förståelse för eller kunskap om OSGi, Sling-komponenter
* Användning av dokumenttjänster
* AEM Designer, AEM workbench, AEM arbetsyta
* Adaptiv formulärkonsol
* Java IDE Tools
* Maven
* GIT/Versionverktyg

<br>

### Mål och omfattning för tentamen {#exam-objectives}

{{exam-objectives}}

**Avsnitt 1: Skapa och hantera Forms (36 %)**

* Bygg Forms
* Konfigurera Forms datamodell
* Anpassa AEM Forms
* Utveckla och använda JavaScript ClientLib med hjälp av GuideBridge API

**Avsnitt 2: Processer och arbetsflöden (34 %)**

* Utforma och genomföra ett arbetsflöde för godkännande av formulär
* Utveckla ett OSGi-paket med Document Services API
* Schemalägg olika asynkrona batchprocesser med Watchfolder-slutpunkt
* Producera olika utdata från XFA/XDP-design
* Hantera PDF med Forms eller Assembler
* Implementera JDBC-tjänster

**Avsnitt 3: Administration och felsökning (18 %)**

* Felsöka och differentiera loggfiler
* Utföra administrationsaktiviteter
* Använd Service Pack
* Utför AEM Dispatcher-konfiguration för Forms
* Navigera mellan olika konsoler

**Avsnitt 4: API-slutpunkter och integreringar (8 %)**

* Integrera med Adobe Experience Cloud
* Generera och förbruka olika slutpunktstjänster

**Avsnitt 5: Interaktiv kommunikation (4 %)**

* Utforma, demonstrera och implementera processer för interaktiv kommunikation
